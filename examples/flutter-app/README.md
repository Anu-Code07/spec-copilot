# Flutter Example

Example SpecDrive project for a Flutter e-commerce app.

See `.specdrive/` for a complete Kiro-style spec:

```
.specdrive/specs/product-review/
├── meta.yaml
├── requirements.md   # EARS user stories
├── design.md         # UI/UX blueprint
└── tasks.md          # Implementation tasks
```

```bash
# When CLI is available (Phase 3)
spec init --stack flutter
spec create "Add product review screen"
```
